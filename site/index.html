A free, open-source monospace font crafted to match Berkeley Mono's aesthetic. Built by configuring Iosevka. Three widths, ten weights, italic variants, and Nerd Font support. Iosevka + Berkeley = Ioskeley.

Installation

  

    

      macOS
      

        
    
          
  1. Download and unzip your chosen file
    2. 
          
  2. Open the width + hint folder that matches your setup (Normal/Hinted/, Normal/Unhinted/, etc.)
    3. 
          
  3. Select all .ttf files, double-click any font, and click Install Font — or drag all into Font Book
    4. 
        

      

    

    

      Windows
      

        
    
          
  1. Download and unzip your chosen file
    2. 
          
  2. Open the width + hint folder that matches your setup
    3. 
          
  3. Select all .ttf files, right-click, and choose Install for all users
    4. 
        

      

    

    

      Linux
      

        
    
          
  1. Download and unzip your chosen file
    2. 
          
  2. Copy the .ttf files to ~/.local/share/fonts/
    3. 
          
  3. Run fc-cache -fv to refresh the font cache
    4. 
        

      

    

    

      What's inside each TTF zip?
      

        

Normal/
  Hinted/     ← standard-DPI (most Windows)
  Unhinted/   ← high-DPI / Retina (macOS, Linux)
SemiCondensed/
  Hinted/
  Unhinted/
Condensed/
  Hinted/
  Unhinted/
        

        
Install all fonts in your chosen folder — your OS exposes the full weight axis (Thin → Black) automatically. Start with Normal/ if you're unsure.

      

    

  


  


  
  
Build from Source

  
The font is built automatically via GitHub Actions on every version tag. To build locally:

  

git clone https://github.com/ahatem/IoskeleyMono.git
git clone --depth 1 https://github.com/be5invis/Iosevka.git

cp IoskeleyMono/private-build-plans.toml Iosevka/
cd Iosevka
npm install
npm run build -- contents::IoskeleyMono contents::IoskeleyMonoTerm
  

  
Output in Iosevka/dist/IoskeleyMono/ and Iosevka/dist/IoskeleyMonoTerm/.
